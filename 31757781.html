Importacia acerca del control de versiones

El control de versiones importante en el desarrollo de software en los siguientesa aspectos:

  • control de versiones

  • gestion de cambios

  • volver a puntos anteriores en el codigo (en caso de errores)

  • optimiza el desarrollor

El control de versiones se ha vuelto un estandar hoy en dia gracias a las buenas practicas, facilidades yherramientas que ofrece git a los desarrolladores, en ves de perder codigo y compartir carpetas git las ordena todas y mantienes historial de los cambios y modificaciones ademas de gestion

Reflexion acerca de la unidad git

Me parecio bastante informativa y util esta unidad de git ya que enseña el optimo manejo de control de versiones en git creacion y fusion de ramas, esto sera de mucha utilidad en el desarrollo de codigo y ayudara a optimizar el desarrollo de una app o en cualquier ambito tecnologico del desarrollo
